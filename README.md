# Nodejs_Assignment
Nodejs_Assignment
  Create the following APIs:

1) Get all products (localhost:3000/products)
2) Get product by id
3) Add product
4) Update product
5) Delete product
6) Create an API that takes the following request:

Request: {
    product_id: 1,
    quantityToBuy: 5
}

Response:
{
        "_id": 1,
        "name": "Product 1",
        "pricePerItem": 10,
        "totalPrice": 50,
        "quantity":5
}
